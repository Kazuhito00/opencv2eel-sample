# opencv2eel_sample
opencv2eel_sampleはEel上でOpenCVの画像を表示するサンプルです。
※EelはElectronライクなHTML/JS/CSSのGUIアプリを作成できるシンプルなライブラリです。

# Requirement
 
* OpenCV 3.4.2
* Eel 0.11.0
 
# Usage
 
サンプルの実行方法は以下です。
WebカメラがPCに接続されている必要があります。
 
```bash
python opencv2eel.py
```
